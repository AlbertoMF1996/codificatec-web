2. Objeto del sitio web

Esta página tiene como finalidad ofrecer información sobre FacturApp, una aplicación de escritorio para la gestión de facturas, clientes, empresas y documentos relacionados.

FacturApp funciona de forma local en el dispositivo del usuario. La información introducida en la aplicación se almacena en el propio equipo del usuario, salvo que este decida exportarla, copiarla, compartirla o enviarla por otros medios.

Codificatec no recibe automáticamente las facturas, clientes, empresas, certificados ni demás información que el usuario introduzca en FacturApp. La aplicación está diseñada para funcionar de forma local en el dispositivo del usuario.

En caso de que el usuario contacte voluntariamente con Codificatec por correo electrónico, los datos facilitados serán utilizados exclusivamente para responder a su consulta, solicitud de soporte o comunicación.
